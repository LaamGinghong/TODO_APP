# TODO APP

>   纯 TypeScript 实现的 MVC 应用

## 什么是 MVC

`MVC` 是组织代码的一种模式，是目前最受欢迎的模式之一：

*   **Model**：管理应用程序的数据
*   **View**：Model 的可视化表示（视图）
*   **Controller**：连接用户和系统



## Model

**model** 就是数据，在当前程序中表示实际的待办事项，以及将会添加、编辑和删除它们的方法。



## View

**view** 是数据的显示方式。在当前程序中，是渲染出来的 HTML。



## Controller

**controller** 连接 **model** 和 **view**。它接受用户输入，比如点击或键入，并处理用户交互的回调。
